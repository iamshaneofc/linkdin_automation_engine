Utility scripts for development, testing, and maintenance.

## Organization

Scripts are organized into subdirectories:

- **migrations/**: Database migration scripts
- **utils/**: General utility scripts
- **tests/**: Test and verification scripts
- **tools/**: Development tools and helpers
- **setup/**: Setup and installation scripts

## Usage

Most scripts can be run directly with Node:

```bash
node scripts/[category]/[script-name].js
```

## Script Categories

### Migrations
Database migration and schema management scripts.

### Utils
General utility scripts for common tasks (data cleanup, exports, etc.).

### Tests
Test scripts for verifying functionality and integrations.

### Tools
Development tools for debugging, monitoring, and analysis.

### Setup
Installation and setup scripts for initial configuration.

## Adding New Scripts

1. Place script in appropriate category directory
2. Add descriptive comments at the top
3. Include usage instructions in comments
4. Update this README if adding new category
